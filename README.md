# HeresTheDrink

HeresTheDrink is a dynamic Android application designed to simplify the exploration and management of beverages. It provides a user-friendly interface, multiple categories, and search functionalities to enhance the user experience.

---

## Features

- **User Authentication**: Login and register functionalities.
- **Categories and Drinks**: Explore drinks by categories.
- **Search and Details**: Search for drinks and view detailed descriptions.
- **Push Notifications**: Firebase Cloud Messaging for user engagement.
- **Responsive UI**: Sleek design using Android layouts and custom drawables.

---

## Technologies Used

- **Language**: Java
- **Framework**: Android SDK
- **Database**: Firebase Realtime Database
- **Messaging**: Firebase Cloud Messaging (FCM)
- **Build Tool**: Gradle
- **UI Assets**: Android XML, custom drawables, and themes.
